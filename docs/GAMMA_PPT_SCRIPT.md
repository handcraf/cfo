## Slide 3: Our Solution

**Title:** AI CFO: Your Private Financial Advisor

**Content:**

✅ **What Team UNSTOPPABLE Built in 36 Hours:**

A fully on-premise AI system that:

1. **Understands** your financial documents (PDF, CSV, XLSX)
2. **Calculates** metrics accurately (code-based, not AI guessing)
3. **Explains** what the numbers mean (natural language)
4. **Answers** questions like a real CFO
5. **Tracks** time periods (Q1, FY, Indian FY Apr-Mar)

🎯 **Key Innovation:**
> "AI explains. Code calculates. Your data stays yours."

**Visual:** Architecture diagram showing Frontend → Backend → Local LLM flow

## Slide 5: Technology Stack

**Title:** Built for Scale, Designed for Privacy

**Content:**

🔧 **Technology Architecture:**

| Layer | Technology | Why |
|-------|------------|-----|
| **Frontend** | React + Vite | Fast, modern UI |
| **Backend** | Go (Golang) | High performance, type-safe |
| **Database** | File-based JSON | Zero dependencies, portable |
| **AI/LLM** | Ollama + LLaMA3 | 100% local, no API keys |
| **Deployment** | Docker Compose | One-command setup |
| **Proxy** | Nginx | Secure routing |

🧠 **AI Model Details:**
- **Model:** Meta LLaMA 3 (8B parameters)
- **Size:** 4.7GB
- **Runs on:** CPU or GPU
- **Latency:** 15-40 seconds per response
- **Context:** 8K tokens

**Built with ❤️ by Team UNSTOPPABLE in 36 hours**

**Visual:** Tech stack diagram with icons for each technology
